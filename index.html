Possibilistic Flux Analysis Toolbox v1.

P. Toolbox for MATLAB is a tool developed by University of Girona and University Polytechnic of Valencia to easily apply Interval MFA and Possibilistic MFA. Interval and Possibilistic MFA are extensions of traditional MFA, which is a widely used procedure to estimate the metabolic fluxes within living cells. These methods, face some limitations that traditional MFA have. Particularly in scenarios of uncertainty —which are indeed common in research laboratories and industry environments—, where measurements are scarce and imprecise.

P Toolbox is:

  • Free.
  • Composed of simple MATLAB functions.
  • Very flexible, users can use in flexible ways and modified.
  • Compatible with COBRA, a well-known toolbox to work with constraint-based metabolic models. Any COBRA-compatible model can be used in P. Toolbox.

P Toolbox allow:

  • Give reliable MFA estimations in uncertain (or underdetermined) scenarios, where only a few fluxes can be measured.
  • Provides MFA estimations accounting for measurements imprecision.

  • Ploting interval estimates and flux distributions.

    As a companion of the MATLAB functions, the P. Toolbox provides a Graphic User Interface (GUI) to represent the flux measurements in possibilistic terms. This has been the most difficult task to new users of Possibilistic MFA.

INSTALLATION

You can download the toolbox here. Download and unzip the archive. Then to install: (1) Copy the PFA Toolbox folder where you want it to be installed. (2) Start MATLAB, (3) Navigate to the PFA toolbox path (4) Run the following script: >> initPFAtoolbox This action will add the PFA Toolbox directories to your MATLAB path and run the initialization script of YALMIP.
