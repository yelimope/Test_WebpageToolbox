Possibilistic Flux Analysis Toolbox v1.

PFA Toolbox for MATLAB is a tool developed by University of Girona and University Polytechnic of Valencia to easily apply Interval MFA and Possibilistic MFA. Interval and Possibilistic MFA are extensions of traditional MFA, which is a widely used procedure to estimate the metabolic fluxes within living cells. These methods, face some limitations that traditional MFA have. Particularly in scenarios of uncertainty —which are indeed common in research laboratories and industry environments—, where measurements are scarce and imprecise.

PFA Toolbox is:

  • Free.
  • Composed of simple MATLAB functions.
  • Very flexible, users can use in flexible ways and modified.
  • Compatible with COBRA, a well-known toolbox to work with constraint-based metabolic models. Any COBRA-compatible model can be used in P. Toolbox.

PFA Toolbox allow:

  • Compute reliable MFA estimations in uncertain (or underdetermined) scenarios, where only a few fluxes can be measured.
  • Calculate MFA estimations accounting for measurements imprecision.
  • Plot interval estimates and flux distributions.
  • Represent easily flux measurements in possibilistic terms with a Graphic User Interface (GUI).

For a better understanding of the tool and methods, we provide an User's manual with several simple examples to illustrate the use of each of the functions. These information is available here.

Download

PFA Toolbox* is a set of MATLAB functions that require solving linear programming problems. To solve these problems, we use a flexible and efficient external optimizer: YALMIP.

You can download PFA Toolbox here by clicking in Download ZIP

Installation

To install the PFA Toolbox:

  1. Unzip PFA-Toolbox file.
  2. Copy the PFA Toolbox folder where you want it to be installed.
  3. Start MATLAB.
  4. Navigate to the PFA-toolbox/Toolbox_Code path
  5. Run the following script: > >>initPFAtoolbox

This action will add the PFA Toolbox directories to your MATLAB path and run the initialization script of YALMIP.
